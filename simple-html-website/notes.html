5 HTML Tags I Learned

Anchor Tag
Used to add a hyperlink using href attribute
Exclamation Mark
Used to set up the html structure of your file
Paragraph
Used to create structure and provides visibility
Abbreviation
Used to provide the complete description/name of an abbreviation when you hover over it
Heading Tags
There are multiple heading tags. We mainly use h1 h2 and h3 to provide structure and titles inside the body.
